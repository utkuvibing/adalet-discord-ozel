**1. Create src/server/middleware/auth.ts** — Socket.IO authentication middleware:

```typescript
import type { Server } from 'socket.io';
import type { ServerToClientEvents, ClientToServerEvents, InterServerEvents, SocketData } from '../../shared/types';
```

Export `registerAuthMiddleware(io: Server<ClientToServerEvents, ServerToClientEvents, InterServerEvents, SocketData>): void`:

- Uses `io.use(async (socket, next) => { ... })` per Socket.IO v4 middleware pattern
- Reads `socket.handshake.auth.token` as string
- If no token AND the connection is from localhost (check `socket.handshake.address` for `127.0.0.1` or `::1` or `::ffff:127.0.0.1`), this is the host — allow through, set `socket.data.displayName = 'Host'`. This lets the host app connect without an invite token.
- If no token and NOT localhost: `next(new Error('MISSING_TOKEN'))`
- If token provided: call `findValidInviteToken(token)` from `../invite`
- If null: `next(new Error('INVALID_TOKEN'))`
- If expired (expiresAt !== null && expiresAt < new Date()): `next(new Error('EXPIRED_TOKEN'))`
- If maxUses reached (maxUses !== null && useCount >= maxUses): `next(new Error('TOKEN_LIMIT_REACHED'))`
- If valid: call `incrementTokenUseCount(invite.id)`, set `socket.data.inviteTokenId = invite.id`, set `socket.data.displayName = socket.handshake.auth.displayName || 'Anonymous'`, call `next()`

**2. Create src/server/signaling.ts** — Socket.IO event handlers:

```typescript
import type { Server, Socket } from 'socket.io';
import type { ServerToClientEvents, ClientToServerEvents, InterServerEvents, SocketData } from '../shared/types';
```

Type alias for convenience:
```typescript
type TypedIO = Server<ClientToServerEvents, ServerToClientEvents, InterServerEvents, SocketData>;
type TypedSocket = Socket<ClientToServerEvents, ServerToClientEvents, InterServerEvents, SocketData>;
```

Export `registerSignalingHandlers(io: TypedIO): void`:

Inside `io.on('connection', (socket: TypedSocket) => { ... })`:

a. **On connect:** Log connection. Send `room:list` with current room state (query rooms from DB, for each room get members from Socket.IO room `room:{id}`, map to RoomWithMembers[]).

b. **room:join** handler:
   - Receive `roomId: number`
   - Leave any current voice room first (iterate `socket.rooms`, find rooms starting with `room:`, leave each, broadcast `system:message` "{displayName} left the room." to that room)
   - Join Socket.IO room `room:{roomId}`
   - Broadcast `system:message` "{displayName} joined the room." to `room:{roomId}`
   - Call `broadcastPresence(io)` to update all clients
   - Emit list of existing peer socket IDs in the room to the joining socket so it can initiate WebRTC connections with each

c. **room:leave** handler:
   - Leave all rooms starting with `room:`
   - Broadcast `system:message` "{displayName} left the room." to left room
   - Call `broadcastPresence(io)`

d. **sdp:offer** relay: Receive `{ to, description }`, emit to target socket ID only: `io.to(payload.to).emit('sdp:offer', { from: socket.id, description: payload.description })`

e. **sdp:answer** relay: Same pattern as offer but with `sdp:answer` event

f. **ice:candidate** relay: Receive `{ to, candidate }`, emit to target: `io.to(payload.to).emit('ice:candidate', { from: socket.id, candidate: payload.candidate })`

g. **disconnect** handler:
   - For each room the socket was in (starting with `room:`): broadcast `system:message` "{displayName} left the room."
   - Call `broadcastPresence(io)`
   - Log disconnection

Helper function `broadcastPresence(io: TypedIO)`:
- Query all rooms from DB using `db.select().from(rooms).all()`
- For each DB room, get Socket.IO room members via `io.sockets.adapter.rooms.get(\`room:${room.id}\`)` (returns Set of socket IDs or undefined)
- For each socket ID in the Set, get the socket via `io.sockets.sockets.get(socketId)` and extract `socket.data.displayName`
- Build `RoomWithMembers[]` array and emit `presence:update` to all connected sockets via `io.emit('presence:update', { rooms: roomsWithMembers })`

Helper function `getPeersInRoom(io: TypedIO, roomKey: string, excludeSocketId: string): string[]`:
- Get socket IDs from `io.sockets.adapter.rooms.get(roomKey)`
- Filter out `excludeSocketId`
- Return array of remaining socket IDs
- Used to send `room:peers` event to newly joined socket listing existing peers to connect to. Add `'room:peers': (peers: string[]) => void` to ServerToClientEvents in types.ts if not already present.
