   - Query the last 50 messages for the room: `db.select().from(messages).where(eq(messages.roomId, roomId)).orderBy(desc(messages.createdAt)).limit(50).all()`
   - For each message row, look up the user to get displayName and avatarId (or join the users table). Use a LEFT JOIN via Drizzle: `db.select({ id: messages.id, roomId: messages.roomId, userId: messages.userId, content: messages.content, createdAt: messages.createdAt, displayName: users.displayName, avatarUrl: users.avatarUrl }).from(messages).leftJoin(users, eq(messages.userId, users.id)).where(eq(messages.roomId, roomId)).orderBy(desc(messages.createdAt)).limit(50).all()`
   - Reverse the array (so oldest first for display), then map to ChatMessage format: `{ id, roomId, userId, displayName: row.displayName || 'Unknown', avatarId: row.avatarUrl || 'skull', content, timestamp: row.createdAt.getTime() }`
   - Emit to the joining socket only: `socket.emit('chat:history', chatHistory)`

3. Add a `chat:message` event handler inside the `io.on('connection')` callback:
   - Validate the socket is in a voice room matching `payload.roomId` (check `socket.rooms.has(ROOM_PREFIX + payload.roomId)`)
   - Validate content is a non-empty string after trimming, max 2000 characters
   - Insert into the messages table: `db.insert(messages).values({ roomId: payload.roomId, userId: socket.data.userId, content: payload.content.trim() }).run()`
   - Get the inserted row ID from the result
   - Construct the ChatMessage object with displayName and avatarId from `socket.data`
   - Broadcast to the entire room (including sender): `io.to(ROOM_PREFIX + payload.roomId).emit('chat:message', chatMessage)`
   - Note: Use `io.to()` not `socket.to()` so the sender also receives the message (canonical rendering from server)
